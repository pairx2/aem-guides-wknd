Input field with Password Strength

Options

  • Password Types : Complex / Moderate
  • Below 4 validations are included :
    1. Min Characters, e.g. 8
    2. Alphabets (uppercase and lowercase), e.g. A-z
    3. Numeric, e.g. 123
    4. Special Character, e.g. #$?
  • For Password type : Complex, all the 4 validations should met to proceed
  • For Password type : Moderate, Min. character and any two out of other three should met to proceed
